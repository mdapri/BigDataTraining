        Jonathan Gao Pavan • a month ago

        It is most likely that you haven't connected to your Azure subscription yet. Please use the instructions in http://azure.microsoft.com/en-...

        There is not an option to stop an HDInsight cluster. The only way is to delete the cluster. Most of the Hadoop jobs are batch processing. It is recommended to delete the cluster after running your job to cut down the cost.

        An easier way to delete a cluster is to use the Azure Management portal. See http://azure.microsoft.com/en-...
